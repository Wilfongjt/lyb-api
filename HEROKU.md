# Heroku Deployment
Heroku Deployment from GitHub

Process 
```
   |
   + <------- (rebase) <----- +
   |                          ^
   |                          |
[Github] ---> (clone) ---> [Development]
   |
(pull request)
   |   
[GitHub Actions]
   |
[Heroku]
```

## Prerequisites
 
1. [GitHub Account](https://github.com)

1. [Heroku Account](https://id.heroku.com/login)


1. Create API_JWT_SECRET
1. Create API_JWT_CLAIMS 
1. Create API_TOKEN
